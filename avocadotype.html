AVOCADOTYPE:

TYPING TEST APPLICATION

AvocadoType is a typing test application that calculates a user’s typing speed. My first full-stack web application, I created this project for my final project while attending Colorado State University’s Coding Dojo Software Development bootcamp.

Users can login/register with an email and password. Once they’re logged in they will be able to test their typing speed using countless phrases.

Users can save their scores, which will be appear on the scores list from highest to lowest.

Users can also find other users on the user list to view their profiles

User profiles display a user’s saved scores and comments to be viewed by other users

The project is based on MERN, a technology stack that utilizes MongoDB document database, the Express.js web framework, React.js client-side framework, and the Node.js server framework.

The front-end client side utilizes API calls that communicate with the server, which contains the API routes, data models, and controllers that post and retrieve information to and from the database.

This project was meant as a proof of concept of programming for a typing test that’s integrated into a full-stack MERN web application with login/registration. Overall, the design is meant to be simple, and needs some polishing. As with any web application, updates, optimizations, and design refreshes are being done over time.
