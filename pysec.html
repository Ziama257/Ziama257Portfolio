PYSEC:

CMD AUTOMATION TOOL

After gaining some foundational Cyber Security knowledge during my Cyber Security Internship at U.S. Department of Homeland Security, I started exploring ways to automate Terminal Commands that are often utilized in network and system security checks.

The user is able to choose from 5 different options, each of which will automatically run netstat or other cmdlets and then return the relevant information.

The tool uses several cmdlets in clever ways to enhance the user’s direct knowledge of their system. For example, this checks which files are immutable on a given drive. Only a few windows system files should be immutable on a regular PC. Outside of these, other files may be malicious.

The tool also allows you to see running tasks, providing the image name and PID. This snapshot can then be used for reference in the use of additional cmdlets to get more information or terminate specific processes.

This project is a Python script that utilizes the subprocess module to run commands with much less effort required by the user. Prompts and returned information are in plain English allowing anyone, not just those experienced in terminal commands, to access these valuable command tools.

The front-end is text-based, featuring ascii art for the title. Navigation through the menu is done by following and keying the on screen prompts.

This project was meant to compile useful shell commands into a single organized tool in a way that makes them more accessible to any user.
